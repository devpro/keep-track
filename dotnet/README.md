## Dependencies

- .NET Core 3.0 SDK
- MongoDB 4.2

## Configuration

- Value for key `KeepTrack_MongoDbConnectionString`: .NET connection string to access MongoDB cluster

## Local run

- Clone the solution: `git clone ...`
- Build the solution: `dotnet build`
